Régression — Prédiction de Valeurs Continues

1. Modèles de Régression
Régression Linéaire
(OLS)
LinearRegression()
  • Simple et rapide
  • Relation linéaire
  • Facile à interpréter
  • Baseline solide
Exemples
  • Prix d'une maison
  • Salaire · Température · Ventes
Régression Régularisée
(L1, L2, Elastic Net)
Ridge() # L2
Lasso() # L1
ElasticNet() # L1+L2
  • Réduit l'overfitting
  • Gère la multicolinéarité
  • Sélection auto des variables (L1)
Types
  • Ridge (L2) : pénalise la taille des coefficients
  • Lasso (L1) : met certains coefficients à 0
  • Elastic Net : combinaison L1 + L2
Régression Polynomiale
PolynomialFeatures()
+ LinearRegression()
  • Transforme les variables pour capturer la non-linéarité
  • Degré du polynôme = complexité du modèle
Exemple
y = β₀ + β₁x + β₂x² + β₃x³ + …
XGBoost Regressor
XGBRegressor()
  • Très performant sur données tabulaires
  • Capture des relations complexes
  • Robuste aux valeurs manquantes (NaN gérés nativement, contrairement à sklearn)
  • Important en pratique

3. Métriques d'Évaluation
MAEMean Absolute Error
(1/n) Σ |yᵢ − ŷᵢ|
Erreur moyenne absolue. Facile à interpréter.
MSEMean Squared Error
(1/n) Σ (yᵢ − ŷᵢ)²
Pénalise fortement les grosses erreurs.
RMSERoot Mean Squared Error
√MSE
Même unité que la variable cible.
R² ScoreCoefficient de détermination
1 − (Σ(yᵢ−ŷᵢ)² / Σ(yᵢ−ȳ)²)
1.0 = parfait · 0 = ne fait pas mieux que la moyenne · < 0 = pire que la moyenne
Comparaison des métriques
MétriqueUtilisation
MAEComprendre l'erreur moyenne
MSEPunir les grosses erreurs
RMSEErreur en unité de la cible
R²Qualité globale du modèle
4. Validation croisée
cross_val_score()
Vérifie la stabilité du modèle sur plusieurs partitions des données. Réduit le risque d'overfitting.
5. Régularisation (L1 / L2)
Ridge (L2)
Pénalise la somme des carrés des coefficients
α Σ βⱼ²
Réduit la variance
Lasso (L1)
Pénalise la somme des valeurs absolues
α Σ |βⱼ|
Sélection de variables (coef = 0)
Elastic Net Combinaison L1 + L2
6. Fonctions Utiles (sklearn / xgboost)
Imports
from sklearn.linear_model import LinearRegression, Ridge, Lasso, ElasticNet
from sklearn.preprocessing import PolynomialFeatures
from xgboost import XGBRegressor
from sklearn.metrics import mean_absolute_error, mean_squared_error, r2_score
from sklearn.model_selection import cross_val_score, train_test_split
Entraînement & Évaluation
model = LinearRegression()
model.fit(X_train, y_train)
y_pred = model.predict(X_test)

mae  = mean_absolute_error(y_test, y_pred)
mse  = mean_squared_error(y_test, y_pred)
rmse = mse ** 0.5
r2   = r2_score(y_test, y_pred)
