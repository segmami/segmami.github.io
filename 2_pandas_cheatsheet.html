Pandas

Reference rapide pour la manipulation
de donnees avec pandas
Import
Importer la librairie
import pandas as pd
Creation de DataFrame
A partir d'un dictionnaire
data = {
  'Nom': ['Nora', 'Ali', 'Sara', 'Karim'],
  'Age': [25, 30, 22, 35],
  'Ville': ['Rabat', 'Fes',
            'Casa', 'Fes']
}
df = pd.DataFrame(data)
Apercu des Donnees
Inspecter le DataFrame
df.head(n) # n premieres lignes
df.tail(n) # n dernieres lignes
df.info() # resume du DataFrame
df.describe() # stats descriptives
df.shape # (lignes, colonnes)
df.columns # noms des colonnes
Selection
Colonnes
df['Nom'] # une colonne (Series)
df[['Nom','Ville']] # plusieurs colonnes
Par position (iloc)
df.iloc[0] # premiere ligne
df.iloc[:, 1] # colonne Age
Par label (loc)
df.loc[0] # ligne d'index 0
df.loc[:,'Ville'] # colonne Ville
Filtrage
Condition simple
df[df['Age'] > 28]
Plusieurs conditions
df[(df['Age'] > 25) &
   (df['Ville'] == 'Fes')]
Avec isin()
df[df['Ville'].isin(
   ['Rabat', 'Fes'])]
Ajout de Colonne
Colonne calculee
df['Annee_naissance'] =
    2025 - df['Age']
Colonne avec valeur fixe
df['Pays'] = 'Maroc'
Tri
Trier les valeurs
df.sort_values('Age') # croissant
df.sort_values('Age',
    ascending=False) # decroissant
df.sort_values(
    ['Ville', 'Age']) # plusieurs cols
Valeurs Manquantes
Detecter & traiter les NaN
df.isna() # detecte les NaN
df.isna().sum() # compte les NaN
df.dropna() # supprime lignes NaN
df.fillna(val) # remplace les NaN
Operations sur Donnees
Calculs & agregations
df['Age'] + 1 # operation colonne
df['Age'] > 30 # comparaison
df['Age'].mean() # moyenne
df.groupby('Ville')['Age'].mean()
Suppression
Supprimer lignes / colonnes
df.drop(columns=['Pays']) # une colonne
df.drop(index=[0]) # une ligne
df.dropna() # lignes avec NaN
Lecture / Ecriture
CSV & Excel
pd.read_csv('fichier.csv') # lire CSV
pd.read_excel('fichier.xlsx') # lire Excel
df.to_csv('f.csv', index=False) # ecrire CSV
df.to_excel('f.xlsx', index=False) # Excel
Important · Sauvegarder les Modifications
La plupart des methodes ne modifient PAS df
df.drop(columns=['Pays']) # df inchange !
df.sort_values('Age') # renvoie une copie
df.dropna() # renvoie une copie
# Le resultat est perdu si non sauvegarde
2 facons de sauvegarder
# 1) reassigner (recommande)
df = df.drop(columns=['Pays'])

# 2) inplace=True
df.drop(columns=['Pays'], inplace=True)
