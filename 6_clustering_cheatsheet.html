Clustering — Apprentissage Non Supervisé

Algorithmes · Métriques internes · Comparaison · Visualisation

Concept

Supervisé : étiquettes connues (y). Non supervisé : pas d'étiquettes — le modèle découvre seul des groupes dans les données.

Objectif Regrouper les points similaires ensemble, séparer les points différents.

Tous les algorithmes reposent sur des distances entre points → la normalisation (StandardScaler / MinMaxScaler) est obligatoire.

K-Means

Diviser en K groupes en minimisant l'inertie (variance intra-cluster). Clusters sphériques, taille similaire, proximité euclidienne.

Méthode du coude (Elbow)
from sklearn.cluster import KMeans

inerties = []
for k in range(2, 11):
    m = KMeans(n_clusters=k,
        random_state=42, n_init=10)
    m.fit(X_scaled)
    inerties.append(m.inertia_)  # coude
+ Rapide, simple, scalable
- Outliers, formes complexes, K manuel
Entraîner avec le meilleur K
kmeans = KMeans(n_clusters=3,
    random_state=42, n_init=10)
labels = kmeans.fit_predict(X_scaled)

print(labels)
print(kmeans.cluster_centers_)  # centroïdes
3 clusters sphériques + centroïdes
Forme attendue du coude (Elbow)
DBSCAN

Identifier des zones denses, marquer les points isolés comme bruit (label -1).

  • eps — rayon de voisinage
  • min_samples — points min pour un cluster
+ Formes complexes, bruit auto, pas de K
- eps difficile, densités variables
Appliquer DBSCAN
from sklearn.cluster import DBSCAN

db = DBSCAN(eps=0.5, min_samples=5)
labels = db.fit_predict(X_scaled)

# -1 = bruit
n = len(set(labels)) - (1 if -1 in labels else 0)
Choisir eps (k-distance)
from sklearn.neighbors import NearestNeighbors
import numpy as np

nbrs = NearestNeighbors(n_neighbors=5).fit(X_scaled)
d, _ = nbrs.kneighbors(X_scaled)
d = np.sort(d[:, -1])
plt.plot(d)  # coude = eps
Densité vs bruit (label -1)
HDBSCAN

Version améliorée de DBSCAN : trouve les clusters les plus stables à travers les échelles de densité, sans fixer eps.

HDBSCAN
import hdbscan  # pip install hdbscan

hdb = hdbscan.HDBSCAN(
    min_cluster_size=10,
    min_samples=5)
labels = hdb.fit_predict(X_scaled)
+ Robuste, densités variables, peu de params
- Plus lent, moins connu
Densités variables gérées
Hiérarchique

Fusionner récursivement les clusters les plus proches selon le linkage, puis couper le dendrogramme.

  • ward — minimise variance (recommandé)
  • complete — distance max
  • average — distance moyenne
  • single — distance min
Dendrogramme + clustering
from sklearn.cluster import AgglomerativeClustering
from scipy.cluster.hierarchy import \
    dendrogram, linkage

linked = linkage(X_scaled, method='ward')
dendrogram(linked, truncate_mode='level', p=5)

h = AgglomerativeClustering(
    n_clusters=3, linkage='ward')
labels = h.fit_predict(X_scaled)
+ Pas de K fixé, dendrogramme
- Lent, sensible au bruit
Dendrogramme — couper à une hauteur
Spectral

Construire un graphe de connexions entre points, puis clusteriser ce graphe. Minimise la coupure.

Spectral Clustering
from sklearn.cluster import SpectralClustering

sp = SpectralClustering(
    n_clusters=3,
    affinity='rbf',
    random_state=42)
labels = sp.fit_predict(X_scaled)
+ Formes non convexes, flexible
- Très lent, K manuel
Formes non convexes (deux lunes)

Métriques — Silhouette
Pas de vérité terrain En clustering, pas d'étiquettes réelles → on utilise des métriques internes qui jugent la qualité des clusters.

Silhouette (la plus utilisée) mesure cohésion + séparation.

s = (b - a) / max(a, b)
  • a = distance moy. dans le même cluster
  • b = distance moy. au cluster voisin
Proche de +1 — bien classé, séparé
Proche de 0 — à la frontière
Proche de -1 — mal classé
silhouette_score + choix de K
from sklearn.metrics import silhouette_score

s = silhouette_score(X_scaled, labels)
print(f"Silhouette : {s:.4f}")

scores = []
for k in range(2, 10):
    km = KMeans(n_clusters=k,
        random_state=42, n_init=10)
    lab = km.fit_predict(X_scaled)
    scores.append(
        silhouette_score(X_scaled, lab))
# le pic = meilleur K
Autres Métriques
Inertie ↓ plus bas = mieux Cohésion intra. Baisse toujours si K monte → Elbow.
Davies-Bouldin ↓ plus bas = mieux Recouvrement inter-cluster. 0 = parfait.
Calinski-H. ↑ plus haut = mieux Variance inter / intra.
Dunn ↑ plus haut = mieux Min dist inter / max diamètre intra.
